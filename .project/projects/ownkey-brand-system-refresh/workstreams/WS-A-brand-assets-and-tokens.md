# Workstream: WS-A Brand Assets and Tokens

## Objective
Turn the brand-book reference into stable Android-ready tokens and assets.

## Owned Files/Areas
- `docs/brandbook/`
- `assets/branding/`
- Android drawable/resource generation targets
- Compose theme/token helpers

## Dependencies

## Risks

## Handoff Criteria
- Token map exists with intended Android usage.
- Brand SVGs are available in source form and converted forms where implementation requires them.
