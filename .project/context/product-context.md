# Product Context

## Users
- Fast typers who value uninterrupted flow
- Multilingual users mixing languages in a single sentence
- Users typing in distinct app contexts (chat, email, notes) with different correction expectations

## Core Flows
- Open keyboard and start typing with instant feedback
- Accept one of top-3 predictions while continuing typing rhythm
- Let autocorrect assist only when confidence is high
- Undo autocorrect immediately when needed
- Maintain personal vocabulary across languages and contexts

## Constraints
- Must preserve user trust by minimizing incorrect corrections
- Must not regress startup/input responsiveness
- Should remain understandable through simple tuning controls
