<p><i>Ownkey Keyboard</i> is an open-source Android keyboard focused on practical AI input with user control.</p>

<p><b>Core focus:</b></p>
<ul>
  <li>Mistral Voxtral ASR as the default voice input path</li>
  <li>Bring your own API key (BYOK)</li>
  <li>Autocorrect and suggestions tuned for real typing</li>
  <li>Clean dark UI with practical customization</li>
</ul>

<p><b>What you can do:</b></p>
<ul>
  <li>Dictate text with Voxtral ASR from inside the keyboard flow</li>
  <li>Set your own endpoint and model in settings</li>
  <li>Use encrypted on-device key storage</li>
  <li>Use updated EN/NL suggestion dictionaries and improved correction ranking</li>
  <li>Customize appearance, key borders, and key corner radius</li>
</ul>

<p><b>Privacy:</b> Ownkey does not add in-app monitoring of private user content. If you enable cloud ASR, requests go to your configured ASR endpoint.</p>

<p><b>Open-source:</b> You can inspect the code and follow development on GitHub.</p>
